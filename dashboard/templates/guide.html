使い方ガイド

サーバー設定の各タブの使い方です。左メニューの「タブ別ガイド」から各項目に飛べます。

📍 設定の開き方
📋 サーバー一覧 → 対象サーバーを開く → 上部のタブ（🔑API / 🌐翻訳 / 🎫チケット …）を切り替えて設定します。各タブは右下の「💾 変更を保存」で保存（転送メッセージのみ追加/編集の瞬間に反映）。

🔑 API設定

Botの動作に必要なキーをまとめて設定します。まず最初にここを設定してください。必要なキーは3つ：① Discord BOTトークン（必須）、② DeepL APIキー（翻訳）、③ Anthropic APIキー（意図判定・翻訳の予備）。以下、それぞれの取得画面と手順です。

🌐 翻訳

指定したチャンネルの英語・日本語の発言を、Botが自動で翻訳して返信します（英文→🇯🇵 / 日本文→🇺🇸）。

  1. 翻訳対象チャンネルを選択（複数可）。チケットカテゴリ全体を対象にもできます
  2. 翻訳プロバイダ（DeepL推奨）と既定の翻訳方向（双方向/英→日/日→英）を選ぶ → 保存
💡 URL・絵文字のみ・短すぎる文は無視。月間予算を超えると自動停止します。

🎫 チケット

お客様が「Open ticket」ボタンを押すと、本人専用のプライベートチャンネルが自動作成されます。

  1. チケット親カテゴリを選択（新規チケットchがここに作られます）
  2. 運営スタッフロールを選択（このロールの人だけがチケットを閲覧可）→ 保存
  3. 「📤 チケットパネルを投稿」で、ボタン付きパネルを任意チャンネルに設置
💡 1ユーザー1チケットまで。🔒クローズで5秒後に削除されます。

🎊 Welcome

新しい人がサーバーに入った瞬間に、歓迎メッセージを自動投稿します。

  1. Welcome投稿先チャンネルを選択
  2. タイトル・本文・色・バナー画像をカスタマイズ → 保存
  3. 「プレビュー」で自分に見え方を確認できます
💡 招待トラッキングと連動し、「どの招待で入ったか」も同時に表示できます。

🤖 意図判定

監視チャンネルの発言をAIで分類し、商品問合せ・配送相談を検知したら誘導文を返信します。

  1. 監視チャンネルと誘導先（#ticket・/shipping 等）を設定 → 保存
  2. ON にすると自動で動作します
💡 反応は同じメッセージに1回だけ。誘導テキストを送るだけで、チケット作成などの能動アクションはしません。Anthropic APIキーが必要です。

📦 送料

商品×数量×宛先国から、DHL/FedExの安い方を自動で計算します（Google Sheets連携）。

  1. スプレッドシート／サービスアカウントを設定
  2. 燃油サーチャージの最新値を確認（毎週自動取得）→ 保存
  3. Discordで /shipping を実行するとカート式パネルが起動
💡 商品重量＋梱包材1kg、0.5kg刻み切り上げ。20kg超は自動で分割発送して合算します。

🏛️ Auction

ボタン入札式オークションの既定値を設定します（実行は Discord の /auction create）。

  1. 最低落札価格・最小増分・アンチスナイプ（終了前の自動延長）等の既定値を設定 → 保存
  2. Discordで /auction create で出品
💡 入札は「現最高値＋5%以上 かつ 最小増分以上」。落札時は落札者×Host×Staffだけの取引チャンネルが自動作成されます（決済は人が調整）。

🚪 招待

入室時にどの招待コードで来たかを自動判定し、Welcome embed と運営ログに記録します。

  1. 判定ログの送信先、/invite create が使えるチャンネル/ロールを設定 → 保存
  2. Discordで /invite list / /invite create
💡 複数Instagramアカウントの流入元判別に。Botに「サーバー管理」権限が必要です。

📨 転送メッセージ

監視元チャンネルの画像＋本文を、別チャンネル（別サーバー可）へ自動転送します。

  1. ① 送信元・② 転送先のチャンネルを選ぶ（サーバー→チャンネルの順）
  2. ③ 送り方（モード）を選ぶ: そのまま／画像のみ／加工（原文＋付加文）／任意テキストに置換
  3. ④ 送信者フィルタ（任意）: 指定ロールの人の投稿だけ転送（未選択なら全員）
  4. 「加工」「任意テキストに置換」ではカスタム文を入力。👀 プレビューで転送先の見え方を確認
  5. 「➕ ルールを追加」で作成。既存ルールは「✏️ 編集」で変更、「🗑 削除」で削除（いずれも即反映）
💡 別サーバーへ転送するには、その転送先サーバーにも転送Botが参加している必要があります。トリガーは「画像を含む投稿」です。

🛠️ /auction create の既定値

ここで設定した値は /auction create の引数を省略したときに使われます。各オークションで個別指定すれば上書き可能です。

最高入札がこの額に満たない場合は 落札なし。`0` = 最低落札価格なし(常に落札)。
5% ルールと両立。両方の高い方が次の必要額になります。
終了 N 秒前以内に入札があれば延長を発動 (既定: 300秒 = 5分)
スナイプ時に追加される秒数 (既定: 30秒)
💡入札ルール
・次の入札は 現最高値 ×1.05 以上 かつ min_increment 以上の上乗せ の両方を満たす必要あり
・例: 現最高値 ¥10,000 / min_increment=100 → 次の必要額は max(10100, ceil(10000×1.05)) = 10,500
・終了 anti_snipe_window 秒以内の入札は自動で anti_snipe_extend 秒延長(繰り返し可)
・通貨は JPY 固定。決済は bot 側で行わない(host が deal channel で個別調整)

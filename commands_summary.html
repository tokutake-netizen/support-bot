A. 翻訳 /translate ・ 英⇄日 自動翻訳

DeepL Free / Anthropic Claude を切替可能。登録チャンネルで自動翻訳。

/translate onAdmin このチャンネルで自動翻訳を有効化
/translate offAdmin このチャンネルの自動翻訳を停止
/translate mode <direction>Admin 方向変更：both / en2ja / ja2en
/translate statusAdmin 有効ch一覧・プロバイダ・月使用量
/translate text <text> [direction]Admin 任意の文字列を即時翻訳して ephemeral 返信
自動動作：英文→🇯🇵、日本文→🇺🇸 を reply。URL／絵文字のみ／2文字未満／bot reply はスキップ。信頼度 TRANSLATE_MIN_CONFIDENCE 未満は無視（既定 0.85）。
🧭
B. 意図判定 /suggest ・ free-talk を Claude で分類

商品問合せ／配送相談を検知し、誘導文を reply（アドバイス止まり）。

/suggest onAdmin 意図判定を有効化
/suggest offAdmin 意図判定を停止
/suggest statusAdmin 現在の有効状態を確認
自動動作：ラベル product_inquiry / shipping_inquiry / general_chat / other。信頼度 ≥ SUGGEST_MIN_CONFIDENCE（既定 0.70）のみ反応。商品問合せ→ #ticket、配送相談→ /shipping ＋ #shipping-guide 誘導。同一メッセージへの reply は1回限り。
📦
C. 送料計算 /shipping ・ /shippingadmin

商品×数量×宛先国 をドロップダウン選択 → Google Sheets から料金取得。

/shipping全員 送料計算パネルを起動（ephemeral）
/shippingadmin reloadAdmin 商品マスタ／スプシキャッシュを再読込
パネル：🎁商品 / 🔢数量 / 🏳️宛先（リージョン→国 or 🔍検索） → 📥送信 → 🧮送料を計算 → 📤公開 / 🔁やり直す / 🌐JP-EN 切替。
計算：合計重量＋梱包1kg → 0.5kg刻みで切上 → 20kg超は自動分割発送。「比較表US基準」の 安い方 列で DHL / FedEx 自動選択。
🎫
D. チケット /ticket ・ Mee6 風サポート起票

ボタンクリックで個別チャンネル作成。1ユーザー1チケットまで。

/ticket panelAdmin このチャンネルにチケット起票パネルを設置
/ticket closeAdmin このチケットチャンネルを閉じる（5秒後削除）
動作：「🎫 チケットを開く」→ Created Tickets 配下に 0001-username 形式で作成。開設者＋運営ロール（TICKET_STAFF_ROLE_IDS）のみ閲覧可。🔒クローズボタンで 5 秒後に自動削除。
👋
E. Welcome /welcome ・ 新規メンバー自動ウェルカム

入室時に Embed を投稿。バナー・色・本文は .env でカスタマイズ。

/welcome testAdmin 自分にプレビュー表示（ephemeral）
/welcome statusAdmin 現在の設定を確認
/welcome setbanner [image] [url]Admin バナー画像を設定（引数なし＝クリア）
自動動作：新規メンバー入室で WELCOME_CHANNEL_ID へ Embed 投稿。WELCOME_AUTOROLE_IDS でロール自動付与。プレースホルダ {user_mention} {guild_name} {member_count} {rules} 等を本文で使用可。
🎉
F. Giveaway /giveaway ・ 時間指定の抽選

参加ボタン式。期限切れで自動抽選＆発表。再抽選・早期終了も対応。

/giveaway create prize duration [winners] [required_role] [image]Admin 新規 Giveaway 開始
/giveaway end <message_id>Admin 早期終了して即座に当選者発表
/giveaway reroll <message_id> [winners]Admin 終了済み抽選の再抽選
/giveaway listAdmin 進行中 Giveaway の一覧
期間フォーマット：30s / 5m / 1h / 1d / 1d2h30m（組み合わせ可）。
参加者：「🎉 参加 / Enter」ボタンを押すだけ。再度押すと取消（トグル）。残量・残り時間は逐次更新。
⚙️
G. 共通仕様 権限・応答形式・エラー表示・ログ

グループを横断する仕様。トラブル時はまずここを確認。

権限：Admin = Administrator 権限保持者のみ。非Adminが打つと Discord 側で弾かれ bot には届かない。/shipping のみ全員実行可。
Ephemeral：管理コマンドは「あなたにのみ表示」。公開は 📤公開する ボタンや /giveaway create 等の明示的な公開コマンド経由のみ。
エラーマーク：⚠️ リアクション = 翻訳 API 失敗（support_bot.log 参照）。❌ メッセージ = 入力エラー（重量超過・該当データなし等）。
ログ：すべての操作・エラーは deployments/<server>/support_bot.log に記録。

🤖 自動アドバイス (意図判定) AI が発言の意図を察知

意図判定機能
Claude で発言を分類 → 商品問合せ/配送相談に該当したら誘導文 reply
0.7 以上の確信度で誘導。低くすると反応が増えるが誤判定も増える。

🚪 オンボーディング

新規メンバーの「初回入室体験」を Discord に組み込みます: ルール表示 → アンケート(最大 5 問) → 各回答に応じてロール自動付与＆チャンネル公開。Discord ネイティブの Onboarding API を直接操作します。

📜 ルール設定

「ルール」と書かれたチャンネルを指定すると、入室画面で「ルールに同意」ステップが表示されます。

このチャンネルに事前にルール本文を投稿しておいてください。

🚪 オンボーディング

オンボーディングを有効化
無効の場合は質問もルール同意も出ません
